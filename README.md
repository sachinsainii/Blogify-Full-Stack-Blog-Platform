# 📝 Blogify-Full Stack Blog Platform

A full-featured Blog Application built using Django that allows users to create, edit, delete, and explore blog posts. 
The application includes authentication, comments, categories, tagging, and admin management.

---

## 🚀 Features

- 🔐 User Authentication (Signup, Login, Logout)
- 👤 User Profile Management
- ✍️ Create, Edit, Delete Blog Posts
- 💬 Comment on Posts
- 🏷️ Categories and Tags
- 🔎 Search and Filter Posts
- 🛠️ Django Admin Panel
- 📱 Responsive Design

---

⚙️ Installation & Setup
 1. Clone the repository
    
    git clone https://github.com/sachinsainii/blogify-Full Stack Blog Platform.git
    
    cd blogify

 3. Create virtual environment
    
    python -m venv venv
    
    venv\Scripts\activate   # Windows

 5. Install dependencies
    
    pip install -r requirements.txt

 7. Run migrations
    
    python manage.py migrate

 9. Start the server
     
    python manage.py runserver
